# Loopstudios Landing Page

## Tech Stack
- HTML5 (Semantic)
- Tailwind CSS v4 (via CDN)
- Vanilla JavaScript

## Dev Server
- Open `index.html` directly in browser, or use Live Server on port 5500

## Design
- Mobile-first approach (375px → 1440px)
- Fonts: Alata (400), Josefin Sans (300)
- Colors: White hsl(0,0%,100%), Black hsl(0,0%,0%), Grey 200 hsl(0,0%,85%)

- `style-guide.md` for design tokens
